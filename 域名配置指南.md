## 1. 在 Vercel 添加域名

### 步骤 1：进入 Vercel 项目设置

1. 打开 [Vercel Dashboard](https://vercel.com/dashboard)
2. 点击您的项目 **ai-code-mentor**
3. 点击 **Settings** 标签

### 步骤 2：找到 Domains 设置

在左侧菜单中，点击 **Domains**

### 步骤 3：添加您的域名

1. 在输入框中输入：`xiaocheng991.site`
2. 点击 **Add**

### 步骤 4：查看 DNS 配置要求

Vercel 会显示需要配置的 DNS 记录，请记录下来（下一步需要用到）

---

## 2. 配置 DNS 解析

### 步骤 1：登录域名服务商

登录您购买域名的平台（阿里云、腾讯云、Godaddy等）

### 步骤 2：找到域名解析设置

1. 进入 **域名管理** 或 **DNS 管理**
2. 找到 **解析记录** 或 **DNS Records**

### 步骤 3：添加 DNS 记录

根据您的域名服务商，选择以下两种方式之一：

#### 方式一：CNAME 记录（推荐）

```
记录类型：CNAME
主机记录：@（或留空）
记录值：cname.vercel-dns.com
TTL：自动 或 600
```

```
记录类型：CNAME
主机记录：www
记录值：cname.vercel-dns.com
TTL：自动 或 600
```

#### 方式二：A 记录（如果不支持 CNAME）

```
记录类型：A
主机记录：@（或留空）
记录值：76.76.21.21
TTL：自动 或 600
```

### 步骤 4：保存配置

点击 **保存** 或 **确认**

---

## 3. 等待域名生效

### 状态说明

在 Vercel 的 Domains 页面，您会看到域名状态：

- ⏳ **Pending**：等待配置（刚添加完）
- ⏱️ **Propagating**：正在生效中
- ✅ **Verified**：已生效，可以访问

### 等待时间

- **通常**：10-30 分钟
- **最慢**：24 小时

### 验证是否生效

```bash
# 在终端输入
ping xiaocheng991.site

# 或者
nslookup xiaocheng991.site
```

如果能解析到 IP 地址，说明已生效。

## ❓ 常见问题

### 1. 域名显示 "Pending" 怎么办？

**原因**：DNS 记录可能还没配置，或者配置有误

**解决**：
1. 检查 DNS 记录是否正确添加
2. 等待 10-30 分钟后再查看
3. 尝试清除本地 DNS 缓存

### 2. 域名解析失败

**原因**：DNS 记录值填写错误

**解决**：
1. 登录域名服务商后台
2. 检查 CNAME/A 记录是否正确
3. 确保没有重复的记录

### 3. 访问显示 "404"

**原因**：域名已生效，但路径不对

**解决**：
确保访问的是完整路径：
```
https://xiaocheng991.site/praise
```

### 4. SSL 证书问题

**解决**：
Vercel 会自动配置 SSL 证书，通常几分钟后会自动生效。
